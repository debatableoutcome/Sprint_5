# Проект автоматизации UI-тестирования сервиса «Доска»

Проект содержит UI-автотесты для учебного сервиса «Доска», написанные на Python с использованием Selenium WebDriver и pytest.  
Тесты покрывают основные пользовательские сценарии работы с сервисом.

---

## Используемые технологии
- Python
- Selenium WebDriver
- pytest
- Google Chrome

---

## Покрываемые сценарии
- Регистрация пользователя:
  - успешная регистрация
  - регистрация с некорректным email
  - регистрация уже существующего пользователя
- Login пользователя
- Logout пользователя
- Создание объявления:
  - неавторизованным пользователем (появление pop-up)
  - авторизованным пользователем (проверка объявления в профиле)

## Установка проекта
1. Склонировать репозиторий или скопировать проект.
2. Создать и активировать виртуальное окружение.
3. Установить зависимости:

pip install selenium pytest

## Запуск тестов

Запуск всех тестов из корня проекта: pytest -v

Запуск отдельного тестового файла: pytest tests/test_registration.py

## Особенности проекта

Все тесты независимы и автономны

Используются только явные ожидания (WebDriverWait)

В тестах отсутствуют sleep, print и неявные ожидания

Все локаторы вынесены в отдельный модуль locators.py

Для тестов регистрации используется генерация уникального email
